REST API consumed by Nogueira Nobre admin dashboard

## Prerequisites
- Node.js installed
- MongoDB installed

## Install

`$ npm install`

## Test

`$ npm test`

## Run

`$ npm start`

Need to set envs:
- AWS_REGION
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
